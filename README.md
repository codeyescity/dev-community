# PFE Project:


## Tech
- Fastapi
- Python
- React
- Html / Css / Js
- Mysql
- And more



## Server Dependencies
Install the dependencies to start the server.

```sh
pip install -r .\requirements.txt
```

## Starting the Server
```sh
python main.py
```

## Client Dependencies
Install the dependencies to start the client.

```sh
npm install
```

## Starting the Client
```sh
npm run dev
```
